## How to Use Custom Block Builder

If you set a page to **"Custom"** template, you can use the advanced block builder:

### **Steps:**

1. Go to `/admin/pages`
2. Edit or create a page
3. **Basic Info Tab:**
   - Template: Select **"Custom"**
4. **Content Tab** (will appear):
   - Click "Add Content Block"
   - Choose block type:
     - **Hero Banner** - Full-width banner
     - **Rich Text** - Formatted text content
     - **Single Image** - Image with caption
     - **Product Grid** - Display products
     - **Custom HTML** - Raw HTML/JS
5. Fill in block content
6. Drag to reorder blocks
7. Save

**Note:** Custom template is for advanced users. For simple pages, use Homepage/About/Contact templates.
